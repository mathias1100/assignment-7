Responsive and Accessibility Findings

At a browser width above 900 pixels, the three exercise sections appear in three columns. Below 900 pixels, the layout changes to two columns, and below 640 pixels it changes to one column. The navigation also wraps when space becomes limited. When I increased the text size to 200% and 400%, the cards became taller, but the text remained readable and no horizontal scrolling was required.

I imported two Google Fonts: Hibur Mono for headings and Atkinson Hyperlegible for the body text. I also added semantic page regions, a skip link, visible keyboard focus, a logical heading order, and relative sizing. The page can be reviewed with the Nu HTML Checker, the document outline, WAVE, keyboard navigation, and a screen reader.
